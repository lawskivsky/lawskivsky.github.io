LCP.11 期望个数统计

原题链接：https://leetcode-cn.com/problems/qi-wang-ge-shu-tong-ji/

题目：

某互联网公司一年一度的春招开始了，一共有 n 名面试者入选。每名面试者都会提交一份简历，公司会根据提供的简历资料产生一个预估的能力值，数值越大代表越有可能通过面试。

小 A 和小 B 负责审核面试者，他们均有所有面试者的简历，并且将各自根据面试者能力值从大到小的顺序浏览。由于简历事先被打乱过，能力值相同的简历的出现顺序是从它们的全排列中等可能地取一个。现在给定 n 名面试者的能力值 scores，设 X 代表小 A 和小 B 的浏览顺序中出现在同一位置的简历数，求 X 的期望。

提示：离散的非负随机变量的期望计算公式为 。在本题中，由于 X 的取值为 0 到 n 之间，期望计算公式可以是 。

拆解题意

1.有两个简历审核官A和B，有n份能力值不同的简历；
2.简历的能力值越高，说明面试者强大，如果我是审核官，我会优先看能力值最大的；
3.

显然，两个审核官都是正常人，都会按照能力值从大到小挑选简历；
假设有5份简历为[1,2,3,4,5],那么，A和B的浏览顺序都为：
A:[5,4,3,2,1];
B:[5,4,3,2,1];
此时，A和B浏览顺序完全相同,根据期望公式可得E(X)=1+1+1+1+1=5。

4.当k份简历中的能力值相同时，期望是多少【可通过概率论求解】:

k=2,则n=[1,1]，此时出现的情况有以下4种：

A B 出现同一位置的简历数
[1,2] [1,2] 2
[1,2] [2,1] 0
[2,1] [1,2] 0
[2,1] [2,1] 2

此时E(X)=(2×A22)/（A22×A22）=1

假设k=3,则n=[1,1,1]，此时出现的情况有36（A33×A33）种，

E（X）=(3×A33+1×A11×A22×C31×C31)/（A33×A33）
